Install 2FlyFlow

Get the app on your phone or computer — no app store needed. It works just like a native app!

✓ 2FlyFlow is already installed on this device!

iPhone / iPad

Safari only — Chrome on iPhone won't work for install

1

Open in Safari

Go to 2flyflow.com in Safari browser

2

Tap the Share button

Tap the ☝ Share icon at the bottom of Safari (square with arrow pointing up)

3

"Add to Home Screen"

Scroll down and tap Add to Home Screen then tap Add

✓

Done! Open from home screen

2FlyFlow icon appears on your home screen. Opens full-screen like a real app!

Android

Works on Chrome, Samsung Internet & Edge

1

Open in Chrome

Go to 2flyflow.com in Chrome

2

Tap the banner or menu

If you see an Install app banner, tap it! Otherwise tap the ⋮ menu (three dots top-right)

3

"Install app"

Tap Install app or Add to Home screen and confirm

✓

Done! Open from home screen

2FlyFlow is now on your home screen and app drawer. Enjoy!

Desktop (Mac / Windows)

Chrome, Edge, Brave or Arc browser

1

Open in Chrome or Edge

Go to 2flyflow.com

2

Click the install icon

Look for the ⊕ Install icon in the address bar (right side) or click the ⋮ menu → Install 2FlyFlow

3

Click Install

Confirm the install popup. The app opens in its own window!

✓

Done! Find it in your apps

2FlyFlow is now in your dock/taskbar and app launcher. No browser chrome, just the app.
